## [描述](https://github.com/sochub/STM8S003/wiki) 

STM8S003工程还包括其相似系列产品，是STM8中最具性价比的系列产品，但由于供货和市场关系，不再是具有性价比的选择。

STM8S003在市场上除了性价比之外，最具有竞争优势就是生态环境已经完善，拥有大量STM的拥趸，习惯了相关的资源开发， 但是对于从8051切换过来的使用者，不是很好的选择。
